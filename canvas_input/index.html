Instructions

After pressing the Save Input button, please move data.json to the canvas_input directory. First, run the following command to generate related files in cache directory.

python prompt_batch.py --prompt-type demo --model gpt-4 --auto-query --always-save --template_version v0.1

Run the folowing command in terminal to generate the images!

python generate.py --prompt-type demo --model gpt-4 --save-suffix "gpt-4" --repeats 5 --frozen_step_ratio 0.75 --regenerate 1 --force_run_ind 0 --run-model lmd_plus --no-scale-boxes-default --template_version v0.1
